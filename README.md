# Titanic Survival Data Analysis

## Project Overview
Exploratory data analysis of the Titanic dataset to identify survival trends among passengers.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Survival analysis by gender, age, and class
- Data visualization for insights
